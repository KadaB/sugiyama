# sugiyama

Graph rendering

## Run
./graphs.py

## Dependencies
- PyGtk 3.0
- PyCairo

Takes graph like:

``
Edges = [(1, 3), (1, 4), (2, 6), (3, 2), (3, 7), (3, 8), (4, 5), (4, 6), (4, 8), (4, 9), (6, 10), (7, 10), (7,     ..11), (8, 7), (9, 11), (10, 12), (11, 12)]
``
